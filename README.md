# WeWine a Full Stack Project by Rachel Fourquet

APIs Used:
  -Facebook OAuth
  -Wine.com
  -Cloudinary
  
Technologies Used:
  -HTML
  -CSS
  -jQuery
  -Node 
  -pSQL

WeWine is a database to search and save your favorites wines. 
You can create a profile, save wines to your personal profile and add comments to each wine.
